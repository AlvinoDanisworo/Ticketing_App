2. Migration dan Model (Konteks MVC)
jadi model itu penghubung kode php dan database
migration itu digunakan untuk bagaimana data data di database bekerja dan terhubung (menentukan struktur dan konfigurasi table)
intinya migration itu kayak nulis query sql, tapi itu dilakukan di php nya, bukan di database nya
ini keywordnya "php artisan migrate"
letaknya di database/migrate dan untuk models ada di app/models

3. SeederÂ 
mengisi table

fungsinya tentunya bedain mana data dummy dan mana data beneran
keywordnya "php artisan make:seeder .....Seeder"
kalua mau jalanin satu satu "php artisan db:seed --class=.....Seeder"
kalua mau langsung semuanya "php artisan db:seed" yang sebelumnya perlu costum file DatabaseSeeder.php dulu

6. Middleware
jadi middleware adalah sebuah komponen khusus di Laravel yang dipake buat perantara antara request dan response
jalan sebelum masuk ke controller
biasanya dipake untuk keamanan (login)

7. Storage Link
jadi di Laravel gambar itu disimpan di dalam folder storage, yang tidak bisa langsung diakses oleh web browser
kita perlu storage link untuk bisa menampilkan gambar
fungsinya adalah sebagai shortcut antara folder storage/app/public dengan public/storage
perintahnya ini "php artisan storage:link"

Jika besok diperlukan saat ujian, karena mengerjakan wajib menggunakan PC lab.
Cara clone project: 
1. buka repository GitHub di browser, copy url repository
2. buat folder baru di komputer, buka di vs code
3. buka terminal, jalankan perintah : git clone (url repository), tunggu sampai selesai
4. masuk ke direktori project, menu file-open folder, pilih folder project
5. buka terminal, jalankan perintah : composer install, tunggu sampai selesai
6. buka terminal, jalankan perintah : npm install, tunggu sampai selesai
7. rename file .env.example menjadi .env, atur konfigurasi database (biasanya di baris 23)
8. buka terminal, jalankan perintah : php artisan key:generate
9. buka terminal, jalankan perintah : php artisan migrate
10. buka terminal, jalankan perintah : php artisan db:seed
11. buka terminal baru, jalankan perintah : php artisan serve
12. buka terminal baru, jalankan perintah : npm run dev
13. jgn lupa php artisan storage:link

disarankan : pakai terminal yang tipenya Command Prompt
